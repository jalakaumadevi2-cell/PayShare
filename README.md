# PayShare - Collaborative Expense Manager

**PayShare** is a web app to simplify group payments and expense sharing. Users can log expenses, assign contributors, and auto-calculate balances.

## Features
- Add/manage shared expenses
- Auto-calculate each member’s share
- Dashboard for total, paid, pending amounts
- Summary chart of expense distribution
- Edit/delete entries dynamically

## Optional Enhancements
- Export summaries to PDF/CSV
- Payment notifications
- Payment API integration (mock)
- Multi-group management

## Folder Structure
PayShare/

## Usage
1. Open `index.html` in a browser
2. Add expense with description, amount, contributors
3. View table and summary of balances
4. Delete any entry dynamically

## Challenges
- Floating-point precision in financial calculations
- Clean UI for quick entry and balance clarity
- Efficient dynamic group data handling
